Mini tutorial: One concrete HTTP message → what FastAPI receives

(req, top_k, authorization)

1) The FastAPI endpoint

from fastapi import FastAPI, Header
from pydantic import BaseModel

app = FastAPI()

class QueryReq(BaseModel):
    question: str

@app.post("/query")
def query(req: QueryReq, top_k: int = 5, authorization: str | None = Header(None)):
    return {
        "q": req.question,
        "top_k": top_k,
        "auth_header": authorization,
    }

2) A real HTTP request message (what the client sends)

POST /query?top_k=10 HTTP/1.1
Host: api.example.com
Authorization: Bearer abc.def.ghi
Content-Type: application/json
Accept: application/json

{ "question": "What does my policy say about refunds?" }

3) How FastAPI maps that message into your function arguments

A) Method + path decide which function runs

  • Method = POST
  • Path = /query

So FastAPI matches: @app.post("/query") → it will call query(...).

B) Query string becomes top_k

From the first line: /query?top_k=10

  • FastAPI finds top_k="10" in the query string
  • Converts "10" → 10 because you typed it as int
  • If it wasn’t provided, it would use the default 5
Result: top_k = 10

C) Header becomes authorization

From the headers: Authorization: Bearer abc.def.ghi

Because you wrote: authorization: str | None = Header(None)

FastAPI reads the Authorization header and sets:

authorization = "Bearer abc.def.ghi"

If the header was missing, it would be None.

D) JSON body becomes req

From the body:

{ "question": "What does my policy say about refunds?" }

Because req is a Pydantic model (QueryReq), FastAPI:

  1. parses the JSON into a dict
  2. validates it matches QueryReq (must have question: str)
  3. constructs a Python object:
req = QueryReq(question="What does my policy say about refunds?")

So inside your function:

  • req is not “the raw request”
  • req is a validated object built from the JSON body
  • you access fields like req.question

4) What FastAPI effectively calls (conceptually)

query(
    req=QueryReq(question="What does my policy say about refunds?"),
    top_k=10,
    authorization="Bearer abc.def.ghi",
)

Summary: That’s the whole mapping: query params → normal args, headers → Header(...) args, JSON body → Pydantic model arg (req), and the method+path choose the endpoint.
