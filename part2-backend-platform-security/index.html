Part II — Backend, Platform & Security

The platform layer that the agent and ML systems sit on top of: FastAPI basics, the request lifecycle, concurrency & user isolation, background work and retries, the full authentication stack (SSO/OAuth2/OIDC/PKCE/JWT vs opaque), workload identity vs end‑user identity, multi‑tenant isolation, external integrations, and the GitHub governance that keeps the codebase shippable.

Reading order tip Chapters 1–5 are the FastAPI runtime story. Chapters 6–10 are the auth story, ordered from OIDC basics through modern sender-constrained patterns. Chapters 11–13 then merge the identity concepts with tenancy, IAM, and GCP workload identity. Chapters 14–15 cover external integrations and GitHub governance.
Prerequisites Comfortable with Python 3, basic HTTP, and the command line. No prior FastAPI knowledge required. The auth chapters (6–15) are theory + code and stand on their own; you do not need to read Part I first.
