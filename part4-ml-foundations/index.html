Part IV — ML Foundations

The geometric, code‑first foundations every later chapter depends on. We start with “what is a model, geometrically?”, then implement linear regression and linear classification end‑to‑end in PyTorch, then cover the tree/ensemble family (Random Forest, AdaBoost, Gradient Boosting, XGBoost) — the family that still beats deep learning on tabular data.

Chapters in order

  1. Introduction to PyTorch
  2. Machine learning as geometry
  3. Linear regression and the training loop
  4. Linear classification (logistic regression)
  5. Trees, bagging & boosting (Random Forest, AdaBoost, XGBoost)
