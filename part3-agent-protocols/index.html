Part III — Agent Protocols & Integration

How modern AI applications connect to tools, data, and other agents. Chapters 1–4 are the MCP track: from first principles, through discovery + remote + auth, into a concrete enterprise pattern, and finally the 2025 spec update. Chapter 5 covers Google's A2A protocol for multi‑agent collaboration. Chapter 6 is a self‑contained JavaScript/TypeScript primer used by Chapter 7's LangChain → LangGraph → Flowise journey.

Prerequisites Comfortable with Python and HTTP basics. Part II Chapters 1–3 (FastAPI & Uvicorn, HTTP methods, request mapping) are recommended before the MCP track. The auth deep‑dive (Part II Ch. 6–9, 15) makes Chapters 2 and 4 easier to follow.
What changed in 2025 MCP moved from HTTP+SSE to Streamable HTTP, formalized authorization on top of OAuth 2.1, introduced elicitation (server‑initiated input requests), structured tool output, and resource links. Chapter 4 is the focused recap; read it right after the original three MCP chapters.
