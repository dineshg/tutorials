# Assets

This directory stores local assets for the legacy static site and the Quarto book source.

## Directories

- `images/`: existing curated local images from the original tutorial.
- `generated/`: deterministic assets generated by `scripts/generate_assets.py`.
- `css/`: legacy static site stylesheet.
- `js/`: legacy static site navigation script.

## Regenerate Generated Assets

```bash
python3 scripts/generate_assets.py
```

Generated assets are original educational diagrams. They do not require private credentials, paid APIs, or remote image downloads.

## Check Asset References

```bash
python3 scripts/check_assets.py
```

Missing local image/media references should block publication.
